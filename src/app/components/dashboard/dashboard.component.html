Nosotros somos FutFinder

FutFinder es una aplicacion echa atravez de Angular para poder ver partidos cercanos a ti y jugar con nuevas personas que necesiten un jugador para una posicion especifica

Venimos aqui para ayudarte a buscar con quien jugar facil y sencillo, atravez de la web con tan solo un click

Nacimos como un aplicativo para una materia de desarrollo web en el año 2023, diseñada por Miguel Fernandez, Gustavo Cascavita y Felipe Rivera.
